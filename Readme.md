# ⚡ Voltwise - Electricity Usage Tracker & Analytics

Voltwise is a full-stack web application designed to help users track, analyze, and visualize electricity consumption using interactive dashboards, graphs, and pie charts.

---

## 🚀 Live Demo
🔗 Click here to view live project 
https://voltwise-app.netlify.app/

---

## 📌 Features

- 🔐 Secure Email OTP Authentication (Gmail SMTP)
- 👤 User Login & Session Handling
- ⚡ Appliance-wise electricity tracking
- 📊 Interactive Graphs for usage analysis
- 🥧 Pie Charts for consumption breakdown
- 📱 Responsive dashboard UI
- 🌐 Fully deployed web application

---

## 🧠 Problem Statement

Many users are unaware of their electricity consumption patterns. Voltwise solves this by converting raw usage data into meaningful insights using visual analytics.

---

## 🛠️ Tech Stack

- HTML5

- SMTP (Gmail OTP Authentication)
